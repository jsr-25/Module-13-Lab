# Module-13-Lab

Project Functionality

This JavaFX application allows a user to securely set a 4-digit PIN with validation and confirmation.

Core Features:
PIN Input
PIN Confirmation

Validation Rules
The application enforces the following checks when the user clicks Set PIN:
PIN must be exactly 4 digits (0–9)
No Repeated Digits
No Sequential Digits
Blocks patterns like 1234 and 4321
PIN Match Check

Output Behavior
If all conditions pass:
Displays: “PIN Set Successfully”
Input fields turn green
If any condition fails:
Displays an appropriate error message
Input fields turn red

Steps to Run the Application:
Using an IDE, locate the main class (e.g., HelloApplication.java)
Run the application
Input two matching pins and click set pin
